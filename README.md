# LXT Clarity

Plateforme interne pour la gestion de la présence, de la disponibilité, des performances et de la visibilité opérationnelle chez LXT.

## Structure

- `backend/` : API FastAPI + logique métier + accès base de données
- `frontend/` : interface utilisateur Next.js
- `docs/` : vision, modèle de données, notes API

## Objectif MVP

- Employés et rôles
- Attendance
- Disponibilité
- Performance
- Projets en cours
- EI tracking
